Repo Management

GitNinja is a suite of applications that provide management utilities and interfaces for configuring and sharing Git repositories and securing access to those repositories.

Enterprise Ready

GitNinja is built with Enterprise features in mind: Expect LDAP integration, workflows, code reviews and utilities to configure and share Git repositories.

Runs on Windows

GitNinja is designed to run on Microsoft Windows, on your enterprise hardware, inside the enterprise firewall. It installs through standard Windows installation mechanisms - no configuration files, promised!

Open Source

Based on .NET, GitNinja will be released and licensed as Open Source software when core development milestones have been reached.
