Mission Statement

  • GitNinja exists to make it easy to collaborate using Git on the Microsoft Windows platform.
  • GitNinja is a suite of applications that provide management utilities and web interfaces for configuring and sharing Git repositories and securing access to those repositories.
  • GitNinja provides out of the box, continuous integration workflows which ease enterprise migration to DVCS from CVCS for software development teams.
  • GitNinja is designed to run on your enterprise hardware, inside the enterprise firewall.
  • GitNinja runs on the Microsoft .Net framework and installs using standard Windows installation mechanisms.
  • GitNinja will be released and licensed as Open Source software when core development milestones have been implemented.
